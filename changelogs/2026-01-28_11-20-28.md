# Changelog

## [Unreleased]

### Removed
- Removed example GitHub Actions workflows for baremetal, Docker, and Kubernetes deployments.

### Added
- Added support for manual CLI usage of MetalDeploy.
- Added documentation for manual CLI usage in `docs/manual-usage.md`.
- Added `--gen-env` and `--sudo` flags to the CLI.
- Added support for overriding configuration using environment variables.

### Changed
- Updated `main.py` to use the `argparse` library for parsing CLI arguments.
- Updated `src/config.py` to load configuration from environment variables and CLI overrides.
- Updated Jenkins integration to use the `ghcr.io/opsguild/metal-deploy:latest` Docker image.
- Updated documentation to reflect changes and additions.
