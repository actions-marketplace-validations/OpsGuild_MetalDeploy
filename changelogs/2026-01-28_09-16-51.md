# Changelog

## [Unreleased]

### Changed
- Improved code formatting and readability throughout the project by adding whitespace and reformatting long lines.
- Updated `src/config.py` to improve the `Config` class initialization and environment variable loading.
- Modified `src/env_manager.py` to enhance environment file generation and parsing logic.
- Refactored `src/git_ops.py` to improve Git authentication and repository cloning.
- Updated `src/orchestrator.py` to enhance deployment handling and error management.
- Improved `src/providers/baremetal.py`, `src/providers/docker.py`, and `src/providers/k8s.py` to refine deployment logic for each provider.
- Enhanced `src/providers/utils.py` to better detect database types and fix permissions.
- Updated test files (`tests/*`) to improve test coverage and reliability.

### Fixed
- Resolved issues with SSH key handling and Git authentication.
- Fixed problems with environment file generation and parsing.
- Addressed errors in deployment scripts for baremetal, Docker, and Kubernetes.
- Corrected database permission issues.

### Removed
- The old `deploy.py` file has been removed and replaced with `main.py`.
- Several test files have been removed, including `test_deploy.py`, `test_env_files.py`, `test_error_handling.py`, `test_git_operations.py`, `test_kubernetes.py`, and `test_ssh_connection.py`.
