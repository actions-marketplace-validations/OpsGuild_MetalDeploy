# Changelog

## [Unreleased]

### Added
- Support for GitHub Secrets (encrypted) and GitHub Variables (plaintext) in environment file generation
- A new section in the README explaining how environment file generation works, including discovery, bucketing, and generation

### Changed
- The default value of `env_files_create_root` has been changed from `true` to `false`
- The entry point of the application has been changed from `deploy.py` to `main.py`
- The `pyproject.toml` file has been updated to include the new `main.py` file and to exclude the old `deploy.py` file

### Removed
- The old `deploy.py` file has been removed and replaced with `main.py`
- Several test files have been removed, including `test_deploy.py`, `test_env_files.py`, `test_error_handling.py`, `test_git_operations.py`, `test_kubernetes.py`, and `test_ssh_connection.py`
